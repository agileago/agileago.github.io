windows下git的安装以及一些问题

发表于 2015-08-15   |  

公司给配了一台PC台式机，因为公司使用git提交代码，然后装上git后发现了一些问题,把这些
问题记录一下,以便以后有机会查阅

  1. 去git官网下载一个git安装包，照着提示安装一下即可
  2. 把git的环境变量配置到PATH路径下面，就是git/bin

  3. 生成ssh文件

  4. 配置git提交时显示的名字

  5. 尝试git clone 一个库，然后会提示你输入yes还是no ,输入yes即可，然后
    一路回车过去

这样配置完成后会发现git的拉取和提交都只能在git bash下面能执行，在cmd下面不能
执行，原来git早就知道这个问题，所以git安装文件下面有一个cmd文件夹，然后把这个文件夹
路径配置到PATH环境变量里面即可，记住这个变量一定要在git/bin变量前面
