js是如何改变开发环境的

发表于 2015-07-16   |  

js创始人始终想不到他无意间创造的一门垃圾语言竟然在默默的改变世界.


我现在的开发环境：

  • 编辑器 Atom
    atom是一款由github所主导的开源的编辑器，使用eletricon和html来开发，其架构相当于打开了本
    地各种权限的chrome浏览器，web下的编辑器。

  • 构建工具 node
    node是基于chrome引擎打造的一套js库，利用它可以作为后端的服务器，也可以作为前端的文件处理
    强有力的助手，如压缩、打包、提示信息等，都由它来驱动

  • 开发语言 js + css + html
    前端开发主要研究领域就是js，研究如何写出更多好用的库，而把一些非常棒的库结合起来就变成了框
    架，再加上html5提供了很多js的api（应该说是新增了很多dom和bom级别的api），让js在前端有了
    无限的可能，CSS3的出现然网页更加充实化，富丽。

  • 调试工具 chrome
    这个浏览器应该是本世纪it行业中最伟大的发明之一，没有V8引擎的驱动，基本上就没有前端如此丰富
    环境，V8无论从效率、性能、体积都完胜同时代的其他浏览器，这也得益于google强大的实力

上面所讲的这些东西基本上构成了一个闭环，以V8引擎为基建，js为上层语言，形成了js开发flow，
开发整个过程一门语言搞定，还忘了说移动端的app,facebook正在做让js开发移动app的无限可能，
这样一来，jser就真的是非常值钱了。

最后安利一下，atom真的很棒！！
